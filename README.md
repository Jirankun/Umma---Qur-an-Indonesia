## 📖 Tentang Umma

**Umma** adalah aplikasi Muslim all-in-one yang dirancang untuk menemani ibadah sehari-hari. Dari Al-Qur'an digital dengan audio murottal, jadwal sholat otomatis, kumpulan doa & dzikir, hingga tracker ibadah harian — semua ada dalam satu aplikasi.

Aplikasi ini **100% gratis tanpa iklan** dan semua data Anda tersimpan aman di perangkat sendiri (tidak ada server database).

---

## ✨ Fitur Unggulan

### 📖 Al-Qur'an Digital
- 114 surah lengkap dengan teks Arab, Latin, dan terjemahan Kemenag RI
- Audio murottal dari **6 Qari** (Sudais, Al-Afasy, dan lainnya) — bisa didownload untuk offline
- Tafsir Kemenag RI per ayat
- Bookmark ayat dengan catatan pribadi
- Program khatam 30 hari dengan target harian
- Mode baca per surah atau per juz

### 🎵 Murattal — Pemutar Audio Quran
- Putar semua 114 surah dengan player di bagian bawah
- Pilih qari favorit, download surah untuk offline
- Download semua surah sekaligus
- Audio tetap jalan walau aplikasi di-minimize atau layar mati
- Kontrol play/pause dari notifikasi Android

### 🤲 Doa & Dzikir
- 10+ kategori doa sehari-hari: sholat, puasa, taubat, perlindungan, dzikir pagi-petang, dan lainnya
- Dilengkapi teks Arab, Latin, arti, dan sumber
- Cari doa, filter kategori, simpan doa favorit

### 📜 Hadits
- 9 kitab hadits shahih (Bukhari, Muslim, Abu Dawud, dan lainnya)
- Hadits Arba'in An-Nawawiyyah (42 hadits) dengan pengaturan tampilan
- Cari hadits berdasarkan teks atau nomor

### ⚖️ Fiqih Islam
- 12 kategori fiqih: Thaharah, Sholat, Puasa, Zakat, Haid, Jenazah, dan lainnya
- 40+ materi lengkap dari Rumaysho.com — bisa diakses tanpa internet

### 💰 Zakat
- 4 kalkulator: Zakat Fitrah, Zakat Maal, Zakat Penghasilan, Zakat Emas & Perak
- Hitung otomatis dengan nisab terkini

### 🧭 Arah Kiblat
- Kompas real-time menggunakan sensor magnetometer + GPS
- Deteksi otomatis apakah sudah menghadap kiblat

### 📊 Tracker Ibadah Harian
- Catat sholat 5 waktu, tarawih, tilawah, puasa, sedekah
- Kalender bulanan dengan statistik dan rata-rata ibadah

### 📝 Jurnal Refleksi
- 4 kategori: Refleksi Harian, Catatan Syukur, Ruang Ikhlas, Catatan Bebas
- Dashboard dengan 5 entri terbaru

### 🤖 Muslim AI
- Tanya jawab seputar Islam dengan AI (5 mode: Ngobrol, Cari Doa, Cari Surah, Tanya Fiqih, Cari Hadits)
- Hasil chat bisa langsung navigasi ke surah/doa/hadits terkait

### Fitur Lainnya

## 🚀 Cara Install

### Download APK
Unduh file APK terbaru dari halaman **[Releases](https://github.com/Jirankun/Umma---Qur-an-Indonesia/releases)** GitHub.

### Build dari Source
```bash
# Clone repositori
git clone https://github.com/Jirankun/Umma---Qur-an-Indonesia.git
cd umma

# Install dependencies
flutter pub get

# Jalankan
flutter run

# Build APK
flutter build apk --release
```

> **Catatan**: Butuh Flutter SDK untuk build dari source. Sebagian besar pengguna cukup download APK dari halaman Releases.

## 🛡️ Privasi & Keamanan

- **100% Local-First** — Semua data disimpan di perangkat Anda, tidak ada server database
- **Tidak Ada Koleksi Data** — Tidak ada data pribadi yang dikirim ke pihak ketiga
- **Transfer Langsung** — Backup file ditransfer langsung antar perangkat via QR Code
- **Tanpa Iklan & Pelacakan** — Aplikasi bersih tanpa iklan, tracker, atau analytics
